## Troubleshooting

### "Not receiving API key"
- Make sure you saved the number (+34 644 34 87 08)
- Send the EXACT message: `I allow callmebot to send me messages`
- Wait 30-60 seconds
- Check your WhatsApp messages

### "Notifications not working"
- Verify API key is correct (no extra spaces)
- Make sure WhatsApp notifications are **enabled** in settings
- Check your mobile number is set in profile
- Mobile number should match the one used to register with CallMeBot

### "API key stopped working"
- API keys don't expire, but check:
- You're using the correct phone number
- WhatsApp is installed and working
- Try requesting a new API key by sending the message again
